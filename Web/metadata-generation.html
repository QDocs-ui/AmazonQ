Message Code Analysis

            

                Script Analysis: Amazon Q can analyze existing 4GL scripts to identify hardcoded messages and automatically generate proper message codes with XML templates.
            

            
            

                
Message Code Format

                

                    

                        Package: tx, tc, td, etc.

                        Table: Table identifier

                        Sequence: 0001, 0002, 0003...
                    

                    

                        Example:

                        txpur400.0001

                        tdsls400.0002
